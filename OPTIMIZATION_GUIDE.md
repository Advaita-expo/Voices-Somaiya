## Recommendations:

### For Profile Images:
- Maximum dimensions: 800x800 pixels
- Format: JPEG with 85% quality
- Target size: < 200KB each

### For the Video:
1. Upload to YouTube as unlisted/public
2. Use YouTube embed code in your website
3. Remove the local MP4 file

### For Background/Banner Images:
- Maximum width: 1920px
- Format: JPEG with 80% quality
- Target size: < 500KB

## Tools for Optimization:
- Online: TinyPNG.com, Squoosh.app
- Software: GIMP (free), Photoshop
- Command line: ImageMagick

## After Optimization:
1. Replace large files with optimized versions
2. Commit to Git repository
3. Push to GitHub successfully
